 Importing Data to Power BI desktop


Steps:

  1. Download the 'Aircraft crash Analysis Data' file by clicking the red icon above
  2. Open Power BI Desktop
  3.  Click on Get data ribbon.
  4.  Select Text/CSV and then click connect
  5. Open saved file, choose 'Aircraft crash Analysis Data' file and click on Open
  6. A preview of the data will be displayed.
