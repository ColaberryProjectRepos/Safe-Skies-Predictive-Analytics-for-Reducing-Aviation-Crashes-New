Steps (Arrow YOY% Crashes)

  1. Click on an empty space on the canvas, and from the visualizations pane select Visual Card
  2. Drag the Arrow YOY % Crashes measure to the fields value box
  3. Fill up figures provided below for the necessary fields
General: 
  • Properties, Size, Height=40, Width=70, Position, Horizontal=138, Vertical=308
  • Title=Off
  •  Effects: Background=Off                
Visual:
  •   Callout Value: Font=DIN (45),
  •    Font color=fx (click on the function tab and fill up values given below)
  •    Format Style = Rules, What field should we base this on? =. YOY% Crashes
  •    Color Codes: Red=#C00000, Green=#079952, If value = is blank then color=#CCCCCCC
