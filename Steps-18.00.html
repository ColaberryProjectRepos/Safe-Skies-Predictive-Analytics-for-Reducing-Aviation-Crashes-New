Creating Slider and Visual Cards for KPIs 18
Steps (Year Slicer)

  1. Click on an empty space on the canvas, and from the visualizations pane select Slider
  2. Under the Fields Pane from Aircraft Crashes Table drag year column to the fields value box
  3. Fill up figures provided below for the necessary fields
General: Properties, Size, Height=111, Width=209, Position, Horizontal=0, Vertical=67
               Title=On, Text=Year, Font=Segoe UI Bold (14), Text color=#FFFFFF, Background color=#0D6ABF, Horizontal alignment=Center
               Effects, Background=On, Background color=#CCCCCC, Visual boarder=On, Border color=#808080

Visual: Values, Font=Segoe UI Bold (10), Font color=#252423
            Slider=On
Steps (Country Slicer)

  1. Copy year slider by press CTRL C and paste it by CTRL V
  2. Under the Fields Pane from Aircraft Crashes Table drag Country column to the fields value box
    3. Follow the GIF to fill up figures provided below for the necessary fields
