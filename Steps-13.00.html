Create Calendar Table & Refresh Date Table 13

Steps: Create Calendar Table 

  1.   On Power BI desktop under Modeling click on new table button
  2.  Type the Dax formula below in the Dax formula bar, click on the check mark, and save.


Steps: Create Refresh Date Table

  1.   On Power BI Desktop ribbon, Click on Transform data, Click on New Source button, and select Blank query from the list and click on Connect
  2.   Under properties in the Name box type in 'Refresh Date'
  3.  On the Dax formula bar type ' =Date Time.LocalNow()'
  4.  Click on Convert to Table button 
  5.  After the refresh table pops up, rename the column by right clicking and          choosing to rename from the selection
  6. Click on Close & apply button and save
