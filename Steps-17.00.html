Adding Project Logo, Title, Image and Refreshed Date for the Dashboard 17
Steps (Colaberry Logo)
  1. From Power BI Desktop ribbon under Insert, click on Image 
  2. Select the Colaberry logo from the downloadable file provided in this project
  3. Under the Visualizations pane, click on General and fill in the figures provided below
          Size: Height=68, Width=207
          Position: Horizontal=10, Vertical=0
Steps (Title)

  1. From Power BI Desktop ribbon under Insert, click on Text box from Elements
  2. Type in 'Aircraft Crash Analysis (1918-2022)' in the text box
  3. Select center alignment for the text (Font = Segoe(bold), Font size = 28, Font Color = #0D6ABF) 
  4.  Click on General and under properties, fill in the figures provided below
              Size: Height=68, Width=659
              Position: Horizontal=383, Vertical=0
  5. Click on Format from Power Bi Desktop, Select Send Backward tab, and click on Send to back button

 

Steps (Aircrash Image)
  1. From Power BI Desktop ribbon under Insert, click on Image 
  2. Select the Aircrash Image from the downloadable file provided in this project
  3. Under the Visualizations pane, Image, Style, and Scaling select the Fill option from the dropdown 
  4.  Click on General and fill in the figures provided below
          Size: Height=68, Width=238
          Position: Horizontal=1043, Vertical=0
Steps: Refreshed Date

  1. Click on an empty space on the canvas, and from the visualizations pane select Visual Card
  2. Drag the Date Time from refresh Date Table to the fields value box
  3. Fill up figures provided below for the necessary fields
