Line and Stacked Column Visual 1(Crashes per Year)

Steps
  1. Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Tab:

  • X axis = Year(groups)
  • Column y axis= Crashes per Year
  • Line Values = Survivor Rate, Fatality Rate
  • Tooltips = Survivor Rate, Fatality Rate, Fatalities, Fatalities per Crash, YOY% Crashes, YOY% Survivor Rate, YOY% Fatality Rate
  • After filling up the above rename fields, as per gif provided below
  • Sort axis by Year Name Short, and ascending order

Format Visual Tab:

General: 
  • Properties, Size, Height=319, Width=551, Position, Horizontal=209, Vertical=68
  •  Title=On, Text=Crashes by Year, Font=Segoe UI Bold (15), Text color=#FFFFFF, Background color=#0D6ABF, Horizontal alignment=Center
  •  Effects: Background=On, Background color=#CCCCCC, Visual boarder=On, Boarder color-#808080

Visual: 
  • X-axis: Values: Font=Segoe UI (9), Font color=#e66c37
  • Title = Off
  •  Y-axis: Values: Font=Segoe UI Bold (9), Font color=#000000, Display         Units=Auto
  •  Title = Off
  •  Secondary Y-axis: Values: Font=Segoe UI Bold (9), Font color=#000000, Display Units=Auto
  •  Title = Off
  • Legend =ON  
  • Gridlines: Horizontal=OFF
  • Columns: Color= #094780
  •  Lines: Colors: Survivor Rate=#e66c37, Fatality Rate=#a38600, Shade area=On
  •  Markers= Off     
  • Data labels:  = Off
