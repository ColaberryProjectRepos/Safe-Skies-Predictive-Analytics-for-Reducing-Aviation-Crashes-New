Data Overview 3
Data Sources:
The dataset used for this project was sourced from Kaggle, a leading platform known for providing high-quality datasets. Kaggle supports data science research by offering a wide range of data across multiple domains, enabling detailed analysis and insights. This platform has been instrumental in facilitating the exploration of aviation safety trends.


System Architecture:
The architecture provides a scalable and robust Power BI solution for data visualization and business intelligence. It incorporates cloud-based modules for data ingestion, preprocessing, and machine learning workflows, enabling real-time data processing. Integration with MS Fabric and dbt enhances data transformation, modeling, and orchestration, supporting complex analytics and ensuring data quality.


Data Feature:
 The Air Crash Analysis Dashboard leverages key data features such as flight phase, crash cause, and weather conditions to analyze crash incidents. It examines patterns over time, identifying risk factors and trends that contribute to crashes. The project transforms this complex data into actionable insights, helping to enhance aviation safety and support data-driven decision-making.
