Creating Waterfall table & Waterfall Chart 22
Steps:
  • In this step we will create a waterfall table by using the sub kpi measures then switch it to a waterfall measure
  • Click on Home tab of power bi ribbon click on the Create Table button then a pop-up window will appear 
  • Add an ID column and add id 1,2, 3,4 for ease of creating the waterfall measure 
  • Add a measure column where we will put our sub KPI measure as they appear in the measure table YoY% Crashes, YOY% Fatalities, YoY% Fatality Rate, YoY% Survivor Rate on the bottom name the table waterfall then saves
  • Click on the water fall you have created and select new measure to create a waterfall measure and insert the DAX formula below 
Waterfall Measure = 
SWITCH (
    SELECTEDVALUE ('Waterfall Table'[ID]),
    1, [. YoY % Crashes],
    2, [.YOY%Survivor Rate],
    3, [. YoY %Fatalities],
    4, [. YoY% Fatality Rate])
Then click the check mark and save.



 Waterfall Chart Visual 4(YOY Change)
 Steps:  
  1.   Right click on the Crashes by Flight Phase visual and copy paste (CTRL+V) 
  2. Click on the water fall chart from the visualization pane
Build Visual Tab:

  • Columns:  Colors Increase #C00000 Decrease ##079952
  • Data Label =ON, Orientation =Horizontal, Position =Inside End 
  • Values: Font=Segoe UI Bold (15), Text color=#FFFFFF
General: Properties Size Height 332 Width 348, Position Horizontal 568 Vertical 387, Title ON Text "YOY Change"
